git init . -> cria um repo git na pasta atual
git status - mostra o status do trabalho (de cada arquivo)
git add nome_arquivo - adiciona arquivos para ser commit
git commit -m "msg" - faz commit do que estava para ser comitado  
git log - mostra todo o histórico de commits


## Fluxo de trabalho 
1. criar duretorio do projeto
2. iniciar o git dentro do diretório criado(git init .)
3. edita ou cria arquivos
4. git status
5. git add <nome do arquivo>
6. git commit -m "mensagem"
7. git status
8. volta para o 3
